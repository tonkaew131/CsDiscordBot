# CsDiscordBot

Discord bot made for Computer Science KMUTNB server

## Getting Started

Clone project
```bash
git clone https://github.com/tonkaew131/CsDiscordBot.git
cd CsDiscordBot
```

Copy [.env.template](.env.template) into `.env` and config
```
cp .env.template .env
```

Install required packages
```bash
npm install
```

Then, run the server:

```bash
npm start
```
