# CsDiscordBot

Discord bot made for Computer Science KMUTNB server

## Getting Started

Clone project
```bash
git clone https://github.com/tonkaew131/CsDiscordBot.git
cd tkw-web
```

Install required packages
```bash
npm install
```

Then, run server:

```bash
npm start
```
